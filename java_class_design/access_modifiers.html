Java Class Design: Access Modifiers

Access modifiers can be casted on classes, variables, constructors and methods

package com.ocpjp;

    public class Main() {
        public int number = 0;

        public Main() { }

        public void publicMethod() {}

There are four access modifiers in Java

  1. Private (Most Restrictive) - accessible from only within the same class.
  2. Default - accessible from only within the same package
  3. Protected - accessible only when subclassed
  4. Public (Least Restrictive) - accessible from anywhere
